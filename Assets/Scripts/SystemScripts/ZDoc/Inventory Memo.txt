[아이템을 얻는 경우]
1. 몬스터 처치 후 드랍된 아이템 획득 => Inventory의 GetItem함수
2. 조합을 통해서 재료를 소모하고 획득 => Inventory의 DefineAction에서 Action 추가 부분


[아이템을 잃는 경우]
1. 아이템 버리기를 할 경우 => UIManager의 OnClickItemJunkBtn()
2. 아이템을 사용할 경우 => UIManager의 OnClickItemUseBtn()
3. 조합으로 재료를 소모할 경우 => CookingManager의 SetData함수에서 Action 추가 부분

-아이템을 잃을 경우 Inveotory의 RemoveItem함수 호출 시 이 함수에서 gm.RemoveItem(id, count)을 하고 실제로 아이템 데이터를 갱신함
